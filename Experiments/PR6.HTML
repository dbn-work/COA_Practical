Practical 6

To simulate a 4x4 memory system using 1x1 chips on COA virtual lab (IIT Kharagpur).

Theory

  1. What is the purpose of using 1x1 memory chips in designing larger memory modules?
  2. Explain the function of a 2-to-4 decoder in memory addressing.
  3. How does grouping memory chips help in achieving desired storage capacity?
  4. Describe the steps to simulate a 4x4 memory system using COA Virtual Lab.

Procedure

  1. Access COA Virtual Lab.
  2. Navigate to memory module design.
  3. Choose 1x1 memory cells.
  4. Create a 2-to-4 decoder for addressing.
  5. Group 4 chips for 4-bit storage.
